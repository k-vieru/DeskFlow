# DeskFlow Quick Start Guide

## Getting Started with DeskFlow

This guide will help you set up and start using DeskFlow effectively.

---

## Step-by-Step Setup

### 1. Create an Account
1. Click **Register** on the login page
2. Enter your:
   - Email address
   - Password (minimum 6 characters)
   - Full name
3. Click **Register**
4. You'll be automatically logged in

### 2. Create Your First Project

**Important**: You must create a project before you can use most features!

#### Option A: From Team Management (Recommended)
1. Go to **Team Management** in the sidebar
2. Click **+ New Project**
3. Enter a project name (e.g., "Website Redesign")
4. Click **Create Project**
5. The project is now available across all tabs!

#### Option B: From Chat
1. Go to **Chat** in the sidebar
2. Click **+ New Project** button
3. Enter a project name
4. Click **Create Project**

### 3. Add Tasks to Your Project

1. Go to **Kanban Board** in the sidebar
2. Select your project from the dropdown
3. Click **+ Add Task** in any column
4. Fill in task details:
   - **Title** (required)
   - **Description** (optional)
   - **Due Date** (optional)
   - **Assign To** (select a team member)
   - **Color** (choose an accent color)
5. Click **Add Task**

### 4. Invite Team Members (Optional)

1. Go to **Team Management**
2. Select your project
3. Click **Invite Member**
4. Enter their email address (they must have an account)
5. Click **Invite**
6. They'll receive a notification and can now access the project

### 5. Log Time on Tasks

Now that you have a project with tasks, you can log time:

1. Go to **Time Logging** in the sidebar
2. The project selector will show your project
3. Click **Log Time** button
4. Select one or more tasks you worked on
5. Enter hours (e.g., 2.5 for 2 hours 30 minutes)
6. Select the date
7. Click **Log Time**

**Troubleshooting**: If you see "Project not found":
- Click the refresh button (‚Üª) next to the project selector
- Make sure you created the project in Kanban Board or Team Management first
- Check the browser console (F12) for detailed error information

### 6. Start Team Chat

1. Go to **Chat** in the sidebar
2. Select your project from the dropdown
3. Type a message and press Enter or click Send
4. Messages sync in real-time for all team members

## Common Workflows

### Daily Workflow for Individual Contributors

1. **Morning**:
   - Check **Notifications** for task completions and updates
   - Review **Kanban Board** to see what needs to be done
   - Check **Chat** for team messages

2. **During Work**:
   - Move tasks through columns (To Do ‚Üí In Progress ‚Üí Done) in **Kanban Board**
   - Use **Chat** to communicate with team
   - Use **Focus Mode** when you need concentration time

3. **End of Day**:
   - Log time spent on tasks in **Time Logging**
   - Update task status in **Kanban Board**
   - Review **Reports** to see your productivity

### Weekly Workflow for Project Owners

1. **Start of Week**:
   - Create new tasks for the week in **Kanban Board**
   - Review **Reports** to check last week's progress
   - Check team capacity and workload distribution

2. **During Week**:
   - Monitor task progress in **Kanban Board**
   - Respond to **Chat** messages
   - Review **Time Logging** stats to track project hours

3. **End of Week**:
   - Generate **Reports** (select "Weekly" view)
   - Export reports if needed
   - Plan next week's tasks

## Feature Overview

### Kanban Board
- **What**: Visual task management with drag-and-drop
- **Who can use**: All project members
- **Special permissions**: Only project owner can add/delete tasks; members can move tasks

### Time Logging
- **What**: Track hours spent on tasks
- **Requirements**: Must have a project with tasks
- **Personal data**: Each user sees only their own time entries

### Chat
- **What**: Team communication per project
- **Real-time**: Messages update every 2 seconds
- **Typing indicators**: See when teammates are typing

### Reports
- **What**: Visual analytics and statistics
- **Views**: Daily, Weekly, or Whole Project
- **Personal vs Team**: See your stats or team-wide data (owners only)

### Team Management
- **What**: Project creation, member invitations
- **Permissions**: Only owners can invite/remove members
- **Voting**: Team votes required to delete projects with tasks

### Notifications
- **What**: Alerts for task completions, invitations, etc.
- **Control**: Enable/disable in Settings
- **Badge**: Shows unread notification count

### Focus Mode
- **What**: Distraction-free environment with timer
- **Pomodoro**: Built-in timer for focused work sessions
- **Music**: Optional background music

### Settings
- **Dark Mode**: Toggle between light and dark themes
- **Notifications**: Enable/disable notifications
- **Profile**: Update your name and email
- **Auto-save**: Settings save automatically on logout

## Error Prevention

### "Project not found" Error
**Prevention**:
1. Always create projects in Kanban Board or Team Management first
2. Use the refresh button if you don't see recent projects
3. Wait a moment after creating a project before using it elsewhere

### "Message text is required" Error
**Prevention**:
1. Type actual text before sending
2. Don't just press Enter on an empty field
3. Make sure you've selected a project

### "No tasks available" Warning
**Prevention**:
1. Create tasks in Kanban Board before trying to log time
2. Make sure you're looking at the correct project

## How to Use

### Accessing Chat
1. Click on "Chat" in the sidebar (icon with speech bubble)
2. Select a project from the dropdown menu
3. Start chatting with your team!

### Sending Messages
1. Type your message in the input field at the bottom
2. Press Enter or click the Send button
3. Your message will appear immediately for all team members

### Keyboard Shortcuts
- **Enter** - Send message
- **Shift + Enter** - New line (not implemented yet, sends message)

## Tips
- Chat history is preserved for each project
- Only project members can see and send messages
- The chat updates automatically every 2 seconds
- Typing indicators update every second for real-time feedback

## Privacy & Access
- You must be a member of a project to access its chat
- Only authenticated users can send and receive messages
- All messages are associated with specific projects
